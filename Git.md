# Git

## show config
```sh
git config --list
```

## config user.name
```sh
git config --global user.name "happy2discover"
```

## config user.email
```sh
git config --global user.email happy2discover@gmail.com
```

## config excludesfile
```sh
git config --global core.excludesfile ~/.gitignore_global
```
